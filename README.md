# L-systems

Poject to program the rules of some famous L-Systems and draw their geographical representations with Python Turtle.

D0I L-System, simple version of an L system.

Axiom: 'B'
Rules: (B -> A), (A -> AB) 


                 b
                 |
                 a
                _|_
               a   b
              _|   \
             a b    a
            _| |    |_
           a b a    a b
          _/  | |_  |_ \   
         a b  a a b a b a
